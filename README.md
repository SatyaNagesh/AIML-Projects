# AIML Projects

A collection of AI/ML portfolio projects to demonstrate full-stack AI engineering skills.

## Projects

| Project | Description | Stack |
|---------|-------------|-------|
| [Autonomous Browser Agent](./autonomous-browser-agent) | AI agent that browses websites, fills forms, and completes multi-step web tasks | LangGraph, OpenAI GPT-5, Playwright, FastAPI, PostgreSQL |
| [Multi-Agent Deep Research Assistant](./multi-agent-research-assistant) | Multi-agent system that researches topics, verifies sources, and generates reports + presentations | LangGraph, OpenAI GPT-5, Tavily, FastAPI, PostgreSQL |
| [AI Interview Copilot](./ai-interview-copilot) | AI-powered interview coach with voice recording, answer evaluation, resume-based questions, and progress tracking | OpenAI GPT-5, Whisper, FastAPI, Next.js, PostgreSQL |
