# AIML Projects

A portfolio of real-world AI/ML projects demonstrating full-stack AI engineering using LangGraph, LangChain, OpenAI, FastAPI, and more.

| # | Project | Stack | Description |
|---|---------|-------|-------------|
| 1 | [Autonomous Browser Agent](https://github.com/SatyaNagesh/AIML-Projects/tree/main/autonomous-browser-agent) | LangGraph, Playwright, FastAPI, PostgreSQL, GPT-5 | Stateful agent that navigates web pages, clicks, fills forms, and extracts structured data via natural language instructions |
| 2 | [Multi-Agent Deep Research Assistant](https://github.com/SatyaNagesh/AIML-Projects/tree/main/multi-agent-research-assistant) | LangGraph, Tavily, FastAPI, PostgreSQL, GPT-5 | Supervisor agent orchestrates 5 specialized sub-agents (Research, FactCheck, SourceCollector, ReportWriter, PresentationCreator) |
| 3 | [AI Interview Copilot](https://github.com/SatyaNagesh/AIML-Projects/tree/main/ai-interview-copilot) | Next.js 15, FastAPI, Whisper, PostgreSQL, GPT-5 | Full-stack app: real-time speech transcription, AI answer evaluation, resume-based question generation, feedback dashboard |
| 4 | [Enterprise RAG Knowledge Assistant](https://github.com/SatyaNagesh/AIML-Projects/tree/main/enterprise-rag-assistant) | LangChain, ChromaDB, FastAPI, PostgreSQL, GPT-5 | PDF/TXT ingestion with vector search and GPT-5 QA with inline source citations |
| 5 | [NL Data Analyst](https://github.com/SatyaNagesh/AIML-Projects/tree/main/natural-language-data-analyst) | OpenAI GPT-5, Pandas, SQL, Plotly, FastAPI, PostgreSQL | Natural language → SQL, automated dataset analysis, interactive charts, dashboards, and HTML reports |

## Getting Started

Each project has its own `docker-compose.yml`, `Makefile`, and `pyproject.toml`. Refer to the project-specific README for setup instructions.
